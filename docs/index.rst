mcu-uuid-telnetd
================

Description
-----------

Microcontroller telnet service

Purpose
-------

Provides access to a console shell as a telnet server (using the
`RFC 854 <https://tools.ietf.org/html/rfc854> protocol`).

Dependencies
------------

* `mcu-uuid-console <https://mcu-uuid-console.readthedocs.io/>`_

Refer to the :download:`library.json <../library.json>` file for more
details.
